Merkle-Tree Issuance

Each citizen becomes a leaf in a binary SHA-256 Merkle tree. Only the root is published - individual leaves stay private.

🛡️

Zero-Knowledge Verification

Third parties verify claims ("age ≥ 18") using ZK proofs. They learn only the statement's truth - nothing more. Not even which leaf is yours.

🔗

Multi-Verifier Unlinkability

Fresh nonce per verification. Even colluding verifiers cannot link proofs to the same citizen. Exactly like Tornado Cash's anonymity set.
